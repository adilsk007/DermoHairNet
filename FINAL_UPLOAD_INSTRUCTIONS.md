# Final GitHub Upload Instructions

Repository name: `dermohairnet-unet`

Repository description:

```text
Deep learning pipeline for dermoscopic hair artifact segmentation and counting using U-Net, OpenCV, and Label Studio annotations.
```

## Upload method

1. Create a new public GitHub repository named `dermohairnet-unet`.
2. Do not initialize it with a README, license, or gitignore.
3. Open this folder.
4. Select all files and folders inside this folder.
5. Drag them into GitHub using `Add file` > `Upload files`.
6. Commit with this message:

```text
Initial commit: add DermoHairNet U-Net dermoscopy hair segmentation project
```

## Do not upload from the old local project folder

Do not upload these files or folders from the old project folder:

```text
.ipynb_checkpoints/
__pycache__/
segmentation.cpython-310.pyc
segmentation.cpython-39.pyc
Untitled.ipynb
Untitled-checkpoint.ipynb
Full Classify folder with generated images
Full data folder with original ISIC images and masks
```

The `Classify`, `data`, and `outputs` folders in this final package are safe placeholder folders.
