# Classify folder

This folder is used for generated image groups after batch classification by predicted hair count.

The original classified output folder was not uploaded because it contained many images and was too large for a clean GitHub repository.

When you run batch classification, generated folders may look like this:

```text
Classify/0/
Classify/1/
Classify/2/
...
Classify/20_plus/
```

Only this placeholder file is tracked in GitHub.
